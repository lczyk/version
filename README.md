# version

Two things in one small package:

1. **`FormatVersion`** — formats a version string from its parts
2. **`generate-version`** — a `go generate` tool that writes those parts from your `VERSION` file and git state

## FormatVersion

```go
import "github.com/lczyk/version"

s := version.FormatVersion(Version, CommitSHA, BuildDate, BuildInfo)
// "0.7.0+5f2fc35 (2026-04-25T10:01:45Z, dirty)"
```

Output format:

```
<version>+<commit7> (<date>, <buildinfo>)
```

Trailing parts are omitted when empty:

| inputs | output |
|---|---|
| version only | `1.2.3` |
| + commit | `1.2.3+abc1234` |
| + date | `1.2.3+abc1234 (2026-01-01T00:00:00Z)` |
| + buildinfo | `1.2.3+abc1234 (2026-01-01T00:00:00Z, dirty)` |

## generate-version

Reads `VERSION` from the project root (walks up from cwd), queries git for the commit SHA and dirty state, and writes a Go file with the four variables.

### Setup

Add a `//go:generate` directive to your version package:

```go
//go:generate go run github.com/lczyk/version/cmd/generate-version -out version.go -pkg version
```

Then run:

```
go generate ./your/version/pkg
```

### Flags

| flag | default | description |
|---|---|---|
| `-out` | `version.go` | output file path |
| `-pkg` | `version` | Go package name |

### Generated output

```go
// Code generated by go generate; DO NOT EDIT.

package version

var (
	Version   = "0.7.0"
	CommitSHA = "5f2fc359fd9dd50384c1fbbc83417924148a5bd4"
	BuildDate = "2026-04-25T10:01:45Z"
	BuildInfo = "dirty"
)
```

Add the generated file to `.gitignore`.

### VERSION file

Plain text, one version per file. Lines starting with `#` and blank lines are ignored:

```
# updated by make release
0.7.0
```
