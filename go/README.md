# version (Go)

Format a version string from its parts. Conforms to [SPEC.md](../SPEC.md).

## FormatVersion

```go
import "github.com/lczyk/version/go"

s := version.FormatVersion(Version, CommitSHA, BuildDate, BuildInfo)
// "0.7.0 @ 5f2fc35 (2026-04-25T10:01:45Z, dirty)"
```

Output format and edge cases: see [SPEC §4](../SPEC.md).

## generate-version

`go generate` tool: reads `VERSION` from project root + git state, writes a Go file with the four named values.

### Setup

Add a `//go:generate` directive to your version package:

```go
//go:generate go run github.com/lczyk/version/go/cmd/generate-version -out version.go -pkg version
```

Then run:

```
go generate ./your/version/pkg
```

### Generated output

```go
// Code generated by go generate; DO NOT EDIT.

package version

var (
    Version   = "0.7.0"
    CommitSHA = "5f2fc359fd9dd50384c1fbbc83417924148a5bd4"
    BuildDate = "2026-04-25T10:01:45Z"
    BuildInfo = "dirty"
)
```

Add the generated file to `.gitignore`.

### Flags

| flag   | default      | description     |
|--------|--------------|-----------------|
| `-out` | `version.go` | output file path |
| `-pkg` | `version`    | Go package name |

### VERSION file

Plain text, one version per file. `#` comments + blank lines ignored. See [SPEC §5](../SPEC.md).
