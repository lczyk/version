# version (Python)

Format a version string from its parts. Mirror of [the Go package](../go/), conforming to [SPEC.md](../SPEC.md).

## format_version

```python
from version import format_version

s = format_version(version, commit_sha, build_date, build_info)
# "0.7.0+5f2fc35 (2026-04-25T10:01:45Z, dirty)"
```

Output format and edge cases: see [SPEC §4](../SPEC.md).

## generate-version

CLI that reads `VERSION` from project root + git state, writes `_version.py`.

Run via `uvx` (no install needed):

```
uvx --from git+https://github.com/lczyk/version#subdirectory=python generate-version
uvx --from . generate-version --out version/_version.py
```

Or installed:

```
pip install .
generate-version
```

Or as a module:

```
uv run python -m version.generate
```

### Generated output

```python
# Code generated by generate-version; DO NOT EDIT.

version = "0.7.0"
commit_sha = "5f2fc359fd9dd50384c1fbbc83417924148a5bd4"
build_date = "2026-04-25T10:01:45Z"
build_info = "dirty"
```

Add the generated file to `.gitignore`.

### Flags

| flag    | default       | description      |
|---------|---------------|------------------|
| `--out` | `_version.py` | output file path |

### VERSION file

Plain text, one version per file. `#` comments + blank lines ignored. See [SPEC §5](../SPEC.md).
